## Списки

Чтобы добавить нненумерованные списки необходимо пункты выделить звездочкой (*). Например вот так:
* Элемент 1
* Элемент 2
* Элемент 3

Что бы добавить нумерованные списки необходимо пункты просто пронумеровать. Например вот так:
1. Первый
2. Второй
